# Clean Kutz | Premium Barber Shop

Precision grooming for modern gentleman. Appointment only.

## Features

- **Premium Design** - Gold (#D4AF37) accents on dark background
- **Custom Cursor** - Gold circular cursor with hover effects
- **Magnetic Buttons** - Follows mouse movement
- **Scroll Reveal** - Elements fade in on scroll
- **Booking System** - Interactive calendar with time slots
- **Service Cards** - 6 services with pricing
- **Gallery** - 4 gallery items with grayscale-to-color hover
- **Testimonials** - Auto-rotating client reviews
- **FAB** - Floating "Book Now" button
- **Loading Animation** - Scissors SVG animation
- **Mobile Menu** - Full-screen mobile navigation

## Services

1. **The Executive** - $45 (45 min) - Precision haircut, hot towel, neck shave
2. **The Classic** - $35 (30 min) - Traditional barber cut with clippers and shears
3. **Royal Shave** - $40 (40 min) - Straight razor shave with hot towels and facial massage
4. **Beard Sculpting** - $30 (25 min) - Shape, trim, and condition with premium beard oil
5. **Young Gentleman** - $25 (30 min) - Haircut for ages 12 and under
6. **The Full Package** - $95 (90 min) - Haircut, beard sculpting, and royal shave

## Technology

- Tailwind CSS (via CDN)
- Lucide Icons
- Google Fonts (Playfair Display, Inter)
- Vanilla JavaScript
- IntersectionObserver for scroll animations
